# Rails Hotwire Todo App

Example repository for the blog post: **[Make server-rendered apps great again with Turbo, Stimulus and view transitions](http://blog.siami.fr/make-server-rendered-apps-great-again)**

A simple todo application built with Rails 8, Hotwire (Turbo + Stimulus), and Tailwind CSS — demonstrating how little code is needed to get smooth, animated page transitions using **Turbo Morphing** and the **View Transitions API**.
